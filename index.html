RoommateHub

GA WDI NYC Jan 14, Project 3

Description

This is an app to help roommates keep track of household bills, events, etc. Users can text the website to get and send information, including adding calendar events, getting the shopping list, and moving items from the shopping list to the bills list.

Technologies Used

  • Ruby 2.1.0
  • Ruby on Rails 4.0.2
  • PostgreSQL Database
  • JavaScript
  • jQuery
  • Backbone
  • Authentication & Authorization using bcrypt-ruby
  • Twilio
